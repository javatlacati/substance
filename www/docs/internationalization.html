Additional languages

You are more than welcome to send the translations of the labels below to kirillcool [at] yahoo.com. Please note:
  • Specify language (ISO 639) and, if relevant, country ( ISO 3166). If you don't know the ISO codes, just write the names in English
  • Provide translations for all the labels.
  • For any non-ASCII character (letters with cedillas, circumflexes etc, non-latin letters) please provide the Unicode code. You can find the relevant charts at Unicode site. Each four-letter code must be preceded by \u.
  • You can use $JAVA_HOME/bin/native2ascii utility to transform native chars to unicode (thanks Claudio).
