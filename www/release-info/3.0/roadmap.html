Roadmap for version 3.0 (Grenada)

Note that the list below is tentative and subject to change
  • Refactor extra feel to a separate project - done in laf-widget project.
  • Improve performance of menu rendering - high. Done.
  • Image-based tool for creating new themes (in theme pack plugin) - low
  • Contextual tabs in ribbon (in ribbon plugin) - low
  • Plugin for JIDE components - low
  • Animation on lists, tables and trees - medium. Done on lists.
  • Support for programmable animated icons on tabs, frames and desktop icons - medium. Done on tabs.
  • Configurable animation for shine spot on buttons - high
  • Additional styles for the title panes - high. Done.
  • API for adding buttons to title panes - high. Done.
  • Improving slider - high. Done.
  • Improving splitter - high. Done.
  • Context menus on text components with Copy / Paste / Cut / Select - high. Done in laf-widget project.
  • Fractal based watermarks (in watermark pack plugin) - high
  • Saturated and desaturated themes - high. Done.
  • Blended themes - high. Done.
  • Grouping of buttons on scrollbars - medium
  • JTabbedPane:
    • Reorderable tabs (with "Recycle bin" ?) - high
    • Detachable tabs - low
    • Tab transition animations - low
    • NB-like popup with all available tabs - low
    • Option for icon-only show - low
    • Ctrl+Tab selection - high. Done in laf-widget project.
    • Support for close buttons on tab overview dialog - high
    • No hover preview of currently selected tab - high. Done in laf-widget project.
