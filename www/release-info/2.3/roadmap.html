Roadmap for version 2.3 (Firenze)

Note that the list below is tentative and subject to change
  • Better support for dark themes - started.
  • Multicolored themes - done.
  • Image-based tool for creating new themes (in theme pack plugin)
  • Contextual tabs in ribbon (in ribbon plugin)
  • Plugin for JIDE components
  • Configurable animation (duration, specify which components don't get it) - done.
  • Animation on lists, tables and trees
  • Animation on selecting / unselecting the checkboxes and radio buttons - done
  • Additional focus style (underline) - done.
  • Support for programmable animated icons on tabs, frames and desktop icons
  • Configurable animation for shine spot on buttons
  • Additional styles for the title panes
  • Image watermark - tiled, frame-bound - done
  • Image watermark - use theme colors - done
  • Toolbar buttons - configure flatness - done.
  • Use current button shaper (if relevant) on tabbed panes and scroll bars. - done.
  • JTabbedPane:
    • Reorderable tabs (with "Recycle bin" ?) and detachable tabs
    • Configuring mouse for one-click tab close (this, all but this, all) - done.
    • Quicktabs
    • Tab transition animations
    • Tab detach
    • NB-like popup with all available tabs
    • Option for icon-only show
    • Tab preview on hover (as with iconified internal frames)
    • Aligning text on left and right tabs - done.
